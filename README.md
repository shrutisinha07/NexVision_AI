# 🖼️ NexVision AI

An AI-powered Image Caption Generator built using Python and the BLIP (Bootstrapping Language-Image Pre-training) model from Hugging Face.

## 🚀 Features

* 📂 Select any image from your computer using a file picker.
* 🤖 Generate AI-powered captions for images.
* 🔁 Caption multiple images without restarting the application.
* 🖼️ Supports JPG, JPEG, and PNG image formats.
* 🐍 Beginner-friendly Python project.

## 🛠️ Technologies Used

* Python
* Transformers (Hugging Face)
* BLIP Image Captioning Model
* PyTorch
* Pillow (PIL)
* Tkinter

## 📁 Project Structure

```
NexVision_AI/
│── NexVision_AI.py
│── README.md
```

## ▶️ How to Run

1. Clone the repository.
2. Install the required libraries.
3. Run:

```bash
python NexVision_AI.py
```

4. Select an image.
5. Let the AI generate a caption.
